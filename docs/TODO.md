## 开发原则 🎯

1. **小步快跑**：每个功能单独分支，独立测试
2. ** backward compatible**：不破坏现有功能
3. **移动优先**：优先考虑移动端体验
4. **性能优先**：避免引入性能瓶颈
5. **渐进增强**：基础功能优先，高级功能逐步添加
