### 2. UI组件库 ✅

| 组件 | 方法 | 说明 |
|------|------|------|
| 玻璃态卡片 | `drawGlassCard()` | 背景+边框+高光 |
| 主要按钮 | `drawPrimaryButton()` | 渐变+发光+按压 |
| 次要按钮 | `drawSecondaryButton()` | 透明+边框 |
| 幽灵按钮 | `drawGhostButton()` | 纯文字 |
| 图标按钮 | `drawIconButton()` | 圆形图标 |
| 数字输入框 | `drawDigitInput()` | 聚焦发光+状态 |
| 键盘按键 | `drawKey()` | 多种变体 |
| 文字标签 | `drawTag()` | 多色变体 |
| 分割线 | `drawDivider()` | 实线/虚线 |
| 列表项 | `drawListItem()` | 图标+文字+详情 |
